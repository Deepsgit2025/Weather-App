🌦️ Weather App

A simple and responsive Weather Application built using **HTML, CSS, and JavaScript**.  
This app allows users to search for any city and view real-time weather information using a weather API.

---

## 📌 Overview

The Weather App fetches live weather data and displays:

- 🌡️ Temperature  
- 🌤️ Weather condition  
- 💧 Humidity  
- 💨 Wind speed  
- 📍 City name  

The application runs entirely in the browser and does not require a backend server.

---

## 🚀 Features

- Search weather by city name
- Real-time weather data
- Clean and responsive UI
- Easy to use
- Lightweight and fast

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Weather API (e.g., OpenWeatherMap)

## 🔑 API Setup (If Required)

If your app uses OpenWeatherMap or another weather API:

1. Create a free account at https://openweathermap.org/
2. Generate your API key.
3. Open `script.js`.
4. Replace the placeholder API key with your own:

```javascript
const apiKey = "YOUR_API_KEY_HERE";
```

5. Save and refresh your browser.
